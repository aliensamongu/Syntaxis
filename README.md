Σyntaxis Pro
Disclaimer: I am not the author of this code, but used AI (Artificial Intelligence) to help me write this program.

Σyntaxis Pro is a highly specialized, offline-first Reed-Kellogg diagramming engine and textual criticism workspace. Designed for linguists and biblical scholars, it combines precision syntax mapping with a built-in manuscript apparatus, all contained within a single, portable HTML/JS file.

🚀 Key Features
Sleek Modern UI: A premium, native-app aesthetic with floating palettes, smooth transitions, and a clean typographic hierarchy.

Smart Word Bank: Automatically generates a draggable word bank from any pasted passage. Words deselect instantly upon placement to keep your workflow fast.

Precision Reed-Kellogg Shapes: Includes Subject baselines, Direct Objects, Modifiers, Appositives, Understood (X) lines, Platform Stilts, and Diamond Compounds.

Dual-Handle Prep Shelf: A custom tool with independent handles for adjusting both the diagonal angle/length and the horizontal shelf width.

Advanced "Single Line" Tool: A freeform line tool with a Right-Click Context Menu to toggle between Solid, Dashed, or Dotted styles and instant Horizontal/Vertical snapping—perfect for manuscript reconstructions.

Critical Apparatus Drawer: A sliding workspace to document manuscript witnesses (e.g., Sinaiticus, Vaticanus, Papyri) directly within your diagram project.

Greek CNTR Smart-Linking: Input a verse reference (e.g., Col 1 or John 1:1) to dynamically generate and launch a link to the corresponding collation data on the Center for New Testament Restoration (CNTR) database.

Local Project Workspace: Save multiple diagrams to your browser's local storage.

Crash Protection: Automatic session recovery ensures your work on the canvas is preserved even if you accidentally refresh the page.

Magnetic Grid Snapping: A toggleable 🧲 system that locks shape anchors to a 10px grid for pixel-perfect alignment.

Vector Exports: Save your diagrams as high-resolution, auto-cropped PNGs or export/import the raw data as JSON files.

⌨️ Controls & Shortcuts
Drag & Drop: Move words from the bank directly onto the canvas.

Marquee Select: Click and drag on the canvas to select multiple elements at once.

Right-Click (Line Tool): Access the styling menu for Single Lines (Solid/Dashed/Dotted and H/V Snapping).

F (or Flip Button): Mirror the selected shape horizontally.

G / U: Group selected elements together / Ungroup them.

Delete / Backspace: Remove selected elements from the canvas.

Arrow Keys: Nudge elements 2px (Shift + Arrows to nudge 10px).

Scroll / Zoom: Use the + and - buttons to navigate large diagrams without losing rendering quality.

🛠 Installation & Usage
Clone or download the repository.

Open index.html in any modern web browser (Chrome, Safari, Firefox, Edge).

No internet connection is required for diagramming or project management; an active connection is only needed to use the external CNTR database links.

External manuscript data integration is provided via the Center for New Testament Restoration (CNTR). We highly recommend their database for users performing deep-dive textual criticism.
