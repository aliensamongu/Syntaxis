Overview
Syntaxis is a specialized software environment designed for the precision diagramming of complex textual
structures. This version (v11.2) is optimized for handling larger blocks of text with high-stability tools for
morphological analysis and notation.

Core Functions & Controls
1. Lexicon Integration (Blue Letter Bible)
Syntaxis can automatically link words to their morphological entries. Hold Option and Click on a word
to open its entry in a new tab.
CRITICAL CAVEAT: For the link system to work, you must use proper capitalization in the reference
box. For example, use "Colossians" or "John" instead of "colossians" or "john." Lowercase book
names will cause the link system to fail.
2. Notation & Footnotes
Hold Option + Cmd/Ctrl and Click on a word to add a footnote.
